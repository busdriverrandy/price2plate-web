# Price2Plate Landing Page

This is a simple, clean, and professional landing page for Price2Plate, styled with Material Design using Materialize CSS.

## Features
- Responsive, modern design
- Clear information about Price2Plate's value
- Ready for GitHub Pages deployment

## How to Deploy on GitHub Pages

1. **Create a GitHub repository** named `price2plate-web` (or your preferred name).
2. **Add the files** from this folder to your repository (at minimum: `index.html`).
3. **Commit and push** your changes to the `main` branch.
4. In your repository, go to **Settings > Pages**.
5. Under "Source," select the `main` branch and `/ (root)` folder.
6. Save. Your site will be live at `https://<your-username>.github.io/price2plate-web/` (or your custom domain if configured).

## Custom Domain
- In GitHub Pages settings, set your custom domain to `price2plate.com`.
- Update your domain's DNS to point to GitHub Pages (see GitHub Docs for details).
